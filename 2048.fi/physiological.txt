The shell commands below use these functions:

    dl(){ local f;for f in "${@:2}";do head -n-"$1" "$f"|sponge "$f";done; }
    ua(){ touch "$1";awk '!a[$0]++' "$1" -|sponge "$1"; }
    st(){ osascript -e'set text item delimiters to linefeed' -e'tell app"safari"to url of tabs of window 1 as text'; }
    scl(){ osascript -e'tell app"safari"to close window 1'; }

The last two functions also work with Chrome if you just change the application name. `sponge` is part of moreutils. The `dl` function requires GNU `head`.

You can use shell commands like this to create a local version of this file and then open and delete the last ten URLs of the file:

    curl -s 2048.fi/physiological.txt|tac|sed -n '/^$/q;p'|tac>physiological
    tail physiological|xargs open;dl 10 physiological
